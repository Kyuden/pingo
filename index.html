Pingo

You will never loose your iPhone again in room.
Pingo provide a pingo command for sounding your iPhone.

Tweet
      
  // Install it yourself as:
  gem install pingo

  // Set it in .zshrc .bashrc etc
  echo "export APPLE_ID=your_id" >> ~/.zshrc
  echo "export APPLE_PASSWORD=your_pass" >> ~/.zshrc
  source ~/.zshrc

  // pingo your_iphone_model_name
  pingo 5s|

Getting started with Pingo

  1. Install pingo gem.
  2. Set AppleID and ApplePass in your shell file.
  3. Use pingo command.
  4. Sound your iPhone :)
